## 🧠 Featured Project

Smart AirVest is an intelligent wearable safety vest designed for real-time environmental and health monitoring.

### Main Features

- Real-time sensor tracking
- Emergency alert system
- MQTT & DDS communication
- Mobile application integration
- Cloud connectivity
- Live monitoring dashboard
- Embedded sensor systems
- Worker safety infrastructure
